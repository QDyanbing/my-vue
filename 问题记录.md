1. 多次调用调用effect的run方法时；要对比更新effect的依赖收集；

- 非第一次调用的时候要做复用；
- 那么为什么不直接记个count，非第一次的时候直接复用；还判断干什么呢；

想了一下大概知道了原因,这样的时候如果只记录数字，而不重新收集以来就会有bug

```js
const flag = ref(true);
const name = ref('John');
const age = ref(20);

effect(() => {
  console.count('effect');
  if (flag.value) {
    app.innerHTML = name.value;
  } else {
    app.innerHTML = age.value;
  }
});
```
